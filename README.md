# Слова
1. Бот пишет первое слово в чат. Генерируеутся последовательность, в которой игроки должны назвать слова.
2. Бот указывает на пользователя, который в течении определенного времени должен назвать слово, которое начинается на последюю букву предыдущего слова.
3. Игрок называет слово и бот должен проверить, что его раньше не было в текущей игровой сессии и оно действительно начинается на нужную букву.
4. Большинство участников чата должны проголосовать, что названное слово - существует.
5. За каждое успешно названное слово начисляются баллы.
6. Игра длится до тех пор, пока не останется 1 игрок.
